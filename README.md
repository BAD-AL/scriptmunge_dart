Replacement 'ScriptMunge' Program written in Dart.

Intended for use on Linux/Steamdeck 

'scriptmunge' depends on 'luac', also included. It needs to be at ./luac or ./bin/luac in order for this scriptmunge program to find in (linux)

The Linux luac is also over at: https://github.com/Gametoast/Lua-5.0.2
